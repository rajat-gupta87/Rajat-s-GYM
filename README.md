🏋️ Rajat’s Gym – Smart Fitness Management System
📌 Overview

Rajat’s Gym is a full-stack web-based fitness management system built using Flask, HTML, CSS, Bootstrap, and PostgreSQL.
It provides secure authentication, admin controls, dashboard analytics, and member progress tracking.

🚀 Features

🔐 User & Admin Authentication

📊 Interactive Dashboard

👥 Member Management

📅 Membership Tracking

📈 Progress & Weight Monitoring

🛠️ Admin Panel Controls

📱 Fully Responsive UI

CSS3

Bootstrap

Backend:

Python

⚙️ Installation
1️⃣ Clone the Repository
git clone https://github.com/your-username/rajats-gym.git
cd rajats-gym
2️⃣ Create Virtual Environment
python -m venv venv
venv\Scripts\activate
3️⃣ Install Dependencies
pip install -r requirements.txt
4️⃣ Setup Environment Variables

Create a .env file and configure:

SECRET_KEY=your_secret_key
DATABASE_URL=your_postgresql_url
5️⃣ Run the Application
python run.py
